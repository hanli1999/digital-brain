## 这是什么

碎片信息丢进收件箱 → AI 自动分类提炼 → 按"洞府"体系归库 → 全局搜索找回。

前端: Vite + React + shadcn/ui · 后端: Hono + Prisma + PostgreSQL · 部署: Vercel + Railway

## 快速开始

```bash
git clone https://github.com/hanli1999/digital-brain.git
cd digital-brain && npm install
npm run dev
```

## 模块

收件箱 · 法器阁 · 功法库 · 丹房 · 机缘录 · AI引擎 · 全局搜索 · Dashboard
